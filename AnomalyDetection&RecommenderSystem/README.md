Low probability ⇒ anomaly Threshold Selection A threshold ε is selected using cross-validation. Prediction rule: if p(x) < ε → anomaly else → normal Evaluation Metric F1 Score: Precision = tp / (tp + fp) Recall = tp / (tp + fn) F1 = (2 * Precision * Recall) / (Precision + Recall) Part 2: Recommender System Objective Build a personalized movie recommendation system using collaborative filtering.
